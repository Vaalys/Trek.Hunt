Welcome to Trek.Hunt!

To use this app, visit http://trek.herokuapp.com/

User Stories: https://trello.com/b/DEkHlwdL/trek-hunt


To get this app up and running on your localhost, follow these instructions:


On Github:

  - Fork this repo and clone to your local repo

In the terminal:

  - bundle install

  - rake db:create

  - rake db:migrate

  - rake db:seed

  - rails s (to run server)

In your browser:

  - go to localhost:3000
